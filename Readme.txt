Adds Full Contact Data To Gravity Forms Entry Detail Page.

== Description ==
Adds Full Contact Data To Gravity Forms Entry Detail Page. Uses the first form field of type 'email' for the full contact person api call.

== Installation ==

1. Upload `fullgravity` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Register at [Full Contact](https://www.fullcontact.com/developer/pricing/) to receive an API key
4. Enter Full Contact API key in Full Gravity Settings
